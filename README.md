# VGHFly（适配超越输入法2.5.1测试版本「1000381」）：

基于原主题修改内容：


1、中/英 键盘样式仿照“小鹤音形-胖版-app”键盘设置。
〉https://flypy.cc/help/#/sj

2、工具栏依次是-菜单-表情-剪贴版-中/英切换-软件本体。

3、新增 全选、复制、粘贴 等一系列快捷操作。

主题使用方法：

1、项目克隆到本地

2、将 VGHFly「键盘主题」 文件夹使用压缩工具进行压缩为.zip

3、在超越输入法中导入该压缩包即可。
